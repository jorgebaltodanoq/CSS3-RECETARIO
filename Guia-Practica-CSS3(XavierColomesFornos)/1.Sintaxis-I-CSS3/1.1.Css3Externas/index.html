DONDE INSERTAR CSS3 DENTRO DE HTML

Podemos insetar nuestro codigo de css de tres maneras:

  • En linea
  • Internas
  • Externas

CODIGO CSS3 NIVEL EXTERNO

Se ingresa en la etiqueta HEAD del codigo HTML y se hace referencia a su ubicacion utilizando la etiqueta LINK. Para este ejemplo vamos a dar estilo de color a todo este documento
