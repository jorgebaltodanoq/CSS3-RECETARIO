DONDE INSERTAR CSS3 DENTRO DE HTML

Podemos insetar nuestro codigo de css de tres maneras:

  • En linea
  • Internas
  • Externas

CODIGO CSS3 EN LINEA

Se ingresa en cada uno de las etiquetas HTML que se requiere mostrar algun estilo, haciendo uso de la etiqueta STYLE. Para este ejemplo vamos a dar estilo de color a todo este documento
